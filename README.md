VERT_stepper updated by jonathan 4/27/21

This file will be used to control vertical stepper motors for now. Here are its commands

A = Speed up
F = Slow down
U = go up
D = go down
M = max speed
B = slowest speed
S = stop

*NOTE: Vertical stepper motors will run at slowest speed upon initiation
